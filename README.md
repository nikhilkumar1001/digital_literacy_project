## 📖 Project Overview
This project was developed as part of the Digital Literacy course. The objective of this project is to understand and apply essential digital skills required in academic and professional environments. 

As a Student Digital Ambassador, I created a Digital Literacy Portfolio that demonstrates knowledge of digital tools, safe online practices, professional communication, and cybercrime awareness. The project consists of five tasks, each focusing on a specific module of digital literacy.

---

## 🧩 Task Details

### 🔹 Task 1: Digital Literacy Infographic
- Created a one-page infographic using **Canva**
- Covered topics:
  - Digital literacy definition  
  - Useful digital tools  
  - Safe internet practices  
  - Professional online presence  
  - Email etiquette  
- Designed using icons, colors, and structured layout for better understanding
